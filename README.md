# Тестовое задание для вакансии
### Скрипт считывает csv-файлы и выводит в консоль таблицу в форме выбранного пользователем отчёта
## Реализованные функциональные требования:
### - пути к файлам передаются параметром --files, поддерживается передача нескольких файлов через пробел
### - выбор отчёта выполняется через параметр --report, пока реализован один отчёт, есть (костыльная) возможность для расширения
### - результат отчёта выводится в консоль
## Прочее:
### - для красивого вывода использована tabulate
### - pandas, numpy не использованы (это было не легко)
### - тесты к критическому функционалу прикручены (это тоже было не легко), проходят, использован pytest 
### - предусмотренна обработка случаев неправльного написания названий параметров, файлов, названий отчётов - показываются подсказки
### - предусмотрены сообщения при несовпадении структур csv-файлов
